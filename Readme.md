        <h1>Git Commands</h1>
        <ol>
            <li>git --version</li>
            <li>git init (creates a git repository)</li>
            <li>git status (check for undone commits)</li>
            <li>git add FILE-NAME (to track file)</li>
            <li>git commit -m "MESSAGE" (to save changes)</li>
            <li>git diff (to see changes in file of Working directory with the last commit)</li>
            <li>git diff --staged (to see changes in file of staged directory with the last commit)</li>
            <li>git commit -a -m "MESSAGE" (a shortcut of doing add and commit in one line)</li>
            <li>git rm --cached FILE-NAME (to remove a file from git's staged repository)</li>
            <li>git branch -M NAME-OF-BRANCH (used to rename a branch)</li>
